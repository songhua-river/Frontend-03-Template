# 组件化
  - 比MVC架构模式更重要，组件化直接决定了前端团队的复用率，提升团队效率
  - 与UI强相关
  - 既是对象又是模块，可以以树形结构来组合，有一定的模板化的配置能力
  
  ## 对象与组件
    - 对象
      - Properties
      - Methods
      - Inherit
    - 组件
      - Properties
      - Methods
      - Inherit
      - Attribute
      - Config & State
      - Event 组件向外传递
      - Lifecycle
      - Children
  
  ## JSX
    - 在代码上的一种快捷方式
    - webpack babel-loader(没有直接依赖babel)
    - @babel/core @babel/preset-env
